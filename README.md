# NexChakra Chatbot
 
## Overview
This is an interactive, button-driven chatbot for the NexChakra website. 
It guides users, suggests demos dynamically, explains solutions, collects leads, and answers FAQs.

## Project Structure
- `backend/` → FastAPI backend
- `frontend/` → Next.js chat UI
- `database/` → demo storage, lead capture, feedback
- `README.md` → project documentation

## Branches
- main
- feature/backend
- feature/frontend
- feature/conversation
- feature/demos
- feature/lead-feedback

## Requirements
- Python 3.10+
- Node.js 18+
- PostgreSQL (or SQLite for local testing)
